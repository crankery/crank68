# Altair 680 BASIC / Micro-soft BASIC for 6800

This is taken from https://gitlab.com/retroabandon/altair-680-basic which was an ambitious project to restore the assembly
source from a binary of Microsoft basic for the Altair 680 computer.

I'm trying to get assembling with vasm6800_oldstyle to run on my 6800 computer.

So, we're starting with a rough disassembly file. Getting this to build to match the binaries it was made from is the first goal.
